This website allows users to search for recipes, manage & maintain favorite recipes and shopping lists, and view recipe preparation methods too.

## Features:
- **Search Recipes:** Search for recipes by name or ingredient.
- **Manage Favorites:** Add recipes to favorites and remove them as needed.
- **Shopping List:** Add ingredients from favorite recipes to a shopping list, displayed in an organized manner.
- **Recipe Methods:** View and toggle the preparation methods for each favorite recipe.
